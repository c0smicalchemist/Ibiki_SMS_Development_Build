Use: setup-postgres.sh

Commands:
```

npm install  # if node_modules missing
PM2_HOME=/home/ibiki/.pm2 pm2 restart ibiki-sms --update-env
npm run db:push --force
```

Best for: Adding PostgreSQL to existing app


OPTION 3: Application Update
-----------------------------
Use: deploy.sh

Commands:
```
cp .env .env.backup
sudo ./deploy.sh
cp .env.backup .env
```

Best for: Updating existing installation
